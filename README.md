文件双端 · 服务端
一个在电脑上运行的 HTTP 文件服务程序。

功能
暴露指定目录下的文件和子目录列表，返回 JSON

支持按名称、大小、修改时间、拍摄日期排序

图片缩略图生成与缓存

视频缩略图生成（需要 FFmpeg）

音频 ID3 元数据读取（标题、艺术家、专辑、封面）

图片 EXIF 信息读取（拍摄日期、GPS）

大文件使用内存映射传输

支持 Range 请求（视频进度拖动）

文本文件分页返回

文本章节自动索引

歌词文件智能匹配与手动映射

文件上传

文件删除

目录创建

自定义封面上传与删除

运行环境
.NET Core 8+

跨平台：Windows / Linux （linux未验证）
启动方式
FileServer.exe
默认监听 http://*:8080。

配置文件 appsettings.json（可选）：

json
{
  "FileServer": {
    "RootPath": "/path/to/your/media",
    "Port": 8080,
    "HttpsPort": 8081,
    "EnableQuic": false
  }
}
额外注意，文件夹需要正确组织，且搜索深度只验证2层级。其他的需要自行判断添加，同时纯粹json（初版顶峰）会带来一定的性能问题
将会在接下来的优化版本中尝试优化各类问题。比如更好的歌词章节匹配轻量据库存储元数据信息等等。注意内存映射。
